HTML controls release 3.2 is now available

Featuring new controls, and new SoHo guidelines.

The Infor HTML controls release 3.2 is available now to all users, with new controls, content matching the guidelines currently found at soho.infor.com, and improvements for accessibility.

Refer to this release as your continuing resource for understanding and implementing the most current SoHo user interface.

BREAKING CHANGES

New Sign In Dialog and About Dialog

Control completely Changed. See new examples

  
     
     
 

Controls Renamed

inforStatusIndicatorChart is now inforBulletChart
inforDualList is now inforSwapList
inforStatusIndicator is now inforProgressIndicator (option statusText is now progressText)
inforLoadingIndicator is now inforBusyIndicator
inforPageLevelMessage is now inforSlideInMessage
inforInlinePopup is now inforPopover

 
     
     

Responsive 3 Column Layout

In the infor3ColumnLayout the container element should be removed from your markup

     
     
 

inforSplashScreen Deprecated

Splash Screens are no longer needed in web applications. For short running tasks use inforBusyIndicator.

 
     
     
 

508 Compliance Markup

  • New Markup for File Input
  • New Markup for Sign In Dialog
  • New class for off screen text inforScreenReaderText
 
     
     
 

Switch Control Markup Change

Change inforSwichLabel from a label to a div and move inforCheckboxLabel up so it is a sibling of inforSwichLabel. A new container label, inforSwitchLabelContainer now wraps the inforSwitchLabel and the inforCheckboxLabel. Remove the "for" attribute and value from the inforSwitchLabel and apply it to the inforSwitchLabelContainer. This change was made so the inforCheckboxLabel could operate the switch.
