BREAKING CHANGES

New Checkboxes

Changed checkboxes to loose weight and be more accessible. On IE7 and IE8 the checkboxes fall back to a regular checkbox as :checked is not supported, but they are still functional. Remove all calls to inforCheckbox()

 
     
     
 

Checkbox Groups no Longer Supported

Checkbox Groups were deprecated and now support is dropped. Use tree control with Checkbox options.

 
     
     

Module Buttons need extra elements

The module buttons (close/settings ect) now need an i element added fx

<button title="Settings" type="button" class="inforModuleSettingsButton">
<i>Settings</i>
</button>

     
     
 

Standalone Nav - Icon and Logo

The App Header Bar for Standalone apps (outside of Mingle) has been changed to allow for a link. See updated markup.
